# Документация Athena LMS

Добро пожаловать в официальную документацию **Athena LMS** — системы управления обучением с открытым исходным кодом. Данный ресурс содержит исчерпывающую информацию по настройке и использованию платформы.

::note{icon="i-simple-icons-github"}
Athena LMS — это Open Source проект. Исходный код, история изменений и трекер задач доступны на [GitHub](https://github.com/shekshuev/athena-lms).
::

## С чего начать

Выберите раздел, соответствующий вашей роли или текущей задаче:

::card-group
  ::card{title="Начало работы" icon="i-lucide-rocket" to="/docs/ru/getting-started"}
  Базовая информация: вход в систему, обзор интерфейса и технические требования.
  ::
  ::card{title="Администраторам" icon="i-lucide-shield-check" to="/docs/ru/admin/users"}
  Руководство по управлению пользователями, ролями, правами доступа и настройками системы.
  ::
::

  ::card{title="Управление пользователями" icon="i-lucide-users" to="/docs/ru/admin/users"}
  Создание учетных записей, блокировка и управление профилями.
  ::
  ::card{title="Ролевая модель" icon="i-lucide-lock" to="/docs/ru/admin/roles"}
  Детальное описание системы прав доступа (RBAC) и политик безопасности.
  ::
  ::card{title="Студия курсов" icon="i-lucide-presentation" disabled="true"}
  *(В разработке)* Инструменты для создания образовательного контента и проверки заданий.

  ::card{title="Обучение" icon="i-lucide-graduation-cap" disabled="true"}
