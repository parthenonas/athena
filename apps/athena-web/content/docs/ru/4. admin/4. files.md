# Файлы и Хранилище

Раздел **Файлы** в панели администратора служит двум целям: модерация загруженного контента и управление лимитами хранилища (квотами) для различных ролей пользователей.

## Файловый менеджер

Таблица отображает файлы, загруженные в систему. Администратор имеет доступ к просмотру и управлению любыми файлами для обеспечения порядка и соблюдения правил.

::card-group
  ::card{title="Уровни доступа" icon="i-lucide-lock"}
  * **Public (Публичный)**: Файлы доступны по прямой ссылке любому пользователю (например, обложки курсов).
  * **Private (Приватный)**: Защищенные файлы, доступные только владельцу и преподавателям (например, ответы на ДЗ).
  ::
  ::card{title="Действия" icon="i-lucide-mouse-pointer-click"}
  Вы можете **Скачать** файл для проверки содержимого или **Удалить** его. Удаление необратимо стирает файл из S3-хранилища.
  ::
::

### Загрузка ассетов
Администраторы могут загружать файлы (например, логотипы или шаблоны). По умолчанию загрузка происходит с правами **Public**.

## Управление Квотами

Athena LMS использует ролевую модель квот для контроля места на диске.



Для настройки нажмите кнопку **:icon{name="i-lucide-database"} Квоты** в правом верхнем углу. Откроется панель управления.

### Политика по умолчанию
Изначально для **всех ролей** в системе установлен лимит **100 MB**. Это гарантирует, что новые роли, созданные в будущем, сразу будут иметь ограничение.

### Настройка исключений
Вы можете переопределить лимит для конкретных ролей (например, выделить Преподавателям 1 GB, а Студентам оставить 100 MB).

::steps{level="4"}
#### Откройте менеджер
Нажмите кнопку **Квоты**. Вы увидите список активных исключений.

#### Добавить правило
1. Нажмите **Добавить правило**.
2. Выберите **Роль** из выпадающего списка.
3. Введите новый лимит в **Мегабайтах (MB)**.
4. Нажмите **Сохранить**.

#### Редактирование и Сброс
* **Изменить:** Введите новое число в поле напротив роли и нажмите иконку сохранения.
* **Сбросить:** Нажмите иконку корзины, чтобы удалить правило. Роль автоматически вернется к дефолтному лимиту **100 MB**.
::
