# Начало работы

Athena LMS — это веб-ориентированная система управления обучением, предназначенная для организации образовательного процесса, управления курсами и контроля успеваемости. Данное руководство содержит инструкции по первичному входу в систему и обзор основных элементов пользовательского интерфейса.

Athena LMS является проектом с открытым исходным кодом. Исходный код доступен на [GitHub](http://github.com/shekshuev/athena-lms).

## Технические требования

Система реализована как веб-приложение. Для корректной работы требуется наличие современного веб-браузера с включенной поддержкой JavaScript.

::warning{icon="i-lucide-triangle-alert"}
**Важно:** Использование устаревших браузеров (например, Internet Explorer) не поддерживается.
::

::card-group
  ::card{title="Google Chrome" icon="i-simple-icons-googlechrome" target="_blank" to="https://www.google.com/chrome/"}
  Рекомендуемый браузер (последняя версия).
  ::
  ::card{title="Firefox" icon="i-simple-icons-firefoxbrowser" target="_blank" to="https://www.mozilla.org/firefox/"}
  Полная поддержка (последняя версия).
  ::
  ::card{title="Safari" icon="i-simple-icons-safari" target="_blank"}
  Для пользователей macOS и iOS.
  ::
::

## Авторизация в системе

Доступ к функционалу Athena LMS ограничен. Учетные записи создаются администраторами.

![Главная страница системы Athena LMS](/images/docs/ru/landing-page-placeholder.png)

::steps{level="4"}
#### Переход к входу
Перейдите на главную страницу и нажмите кнопку **НАЧАТЬ ОБУЧЕНИЕ** (или **ВХОД** в углу).

#### Ввод данных
Вы попадете на страницу `/auth/login`. Заполните поля:

::field-group
  :::field{name="Логин" type="text" required="true"}
  Ваш уникальный идентификатор (не Email). Пример: `ivanov_ii`.
  :::
  :::field{name="Пароль" type="password" required="true"}
  Выдается администратором.
  :::
::

#### Подтверждение
Нажмите кнопку **Продолжить**.
::

::note{icon="i-lucide-info"}
Если вы забыли пароль или видите ошибку *Invalid credentials*, обратитесь к администратору. Самостоятельный сброс пароля пока не предусмотрен.
::

## Обзор интерфейса

После входа основным инструментом навигации становится **Боковая панель**.

::card-group
  ::card{title="Обучение (Learn)" icon="i-lucide-book-open"}
  Раздел для студентов. Курсы, расписание, файлы и сообщество.
  ::
  ::card{title="Студия (Studio)" icon="i-lucide-presentation"}
  Раздел для преподавателей. Создание курсов, проверка заданий и аналитика.
  ::
  ::card{title="Администрирование" icon="i-lucide-shield-check"}
  Только для администраторов. Управление пользователями, ролями и настройками системы.
  ::
::

### Меню пользователя

Внизу боковой панели (клик по аватару) доступны:
* **Настройки:** Профиль пользователя.
* **Выйти:** Безопасное завершение сеанса.

::caution{icon="i-lucide-shield"}
Всегда нажимайте **Выйти** при завершении работы на общедоступных компьютерах.
::
