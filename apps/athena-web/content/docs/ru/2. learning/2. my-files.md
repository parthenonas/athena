# Личное Хранилище

Раздел **Мои файлы** — это ваше персональное облачное пространство внутри Athena LMS. Сюда можно загружать домашние задания, документы для проектов и другие материалы, необходимые для учебы.

## Использование места

В верхней части страницы расположен виджет состояния хранилища.

::alert{icon="i-lucide-hard-drive"}
Прогресс-бар показывает, сколько места занято и каков ваш общий лимит (Квота). Если вы достигнете **100%**, загрузка новых файлов будет заблокирована, пока вы не освободите место.
::

Если вам не хватает места, обратитесь к преподавателю или администратору.

## Управление файлами

### Загрузка
1. Нажмите кнопку **Загрузить файл**.
2. Перетащите файл в зону загрузки или выберите его на компьютере.
3. **Доступ:** По умолчанию файлы загружаются как **Private** (приватные). Их видите только вы и преподаватели. Если нужно поделиться файлом с другими студентами, можно переключить на **Public**.

### Действия с файлами
В списке доступны следующие действия:
* **:icon{name="i-lucide-download"} Скачать:** Сохранить файл на устройство.
* **:icon{name="i-lucide-link"} Копировать ссылку:** Получить ссылку для обмена (только для *Public* файлов).
* **:icon{name="i-lucide-trash"} Удалить:** Безвозвратно удалить файл, чтобы освободить место в квоте.

## Форматы
Система поддерживает большинство популярных форматов документов и изображений (PDF, DOCX, JPG, PNG).
