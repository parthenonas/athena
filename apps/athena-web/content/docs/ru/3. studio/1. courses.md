# Управление курсами

**Course Manager** (Менеджер курсов) — это главный инструмент для авторов и преподавателей. Здесь вы создаете структуру обучения, управляете жизненным циклом курсов (Черновик / Опубликован) и переходите в конструктор контента.

## Обзор

На главной странице отображается список всех курсов, которые вы создали или к которым имеете доступ.

::card-group
  ::card{title="Черновики (Drafts)" icon="i-lucide-pencil"}
  Курсы, находящиеся в разработке. Они **не видны** студентам в разделе обучения.
  ::
  ::card{title="Опубликованные (Published)" icon="i-lucide-globe"}
  Готовые курсы, доступные для изучения студентами.
  ::
::

Переключаться между списками можно с помощью вкладок в верхней части страницы.

## Создание курса

Чтобы начать, нажмите кнопку **Создать курс**.

::steps{level="4"}
#### Основная информация
Введите **Название** курса. Это единственное обязательное поле для старта.

#### Описание и Теги
Добавьте описание, чтобы студенты понимали суть курса. Также можно добавить **Теги** (например, `Математика`, `JS`, `Сложно`), чтобы упростить поиск.
*Чтобы добавить тег, введите текст и нажмите Enter.*

#### Сохранение
Нажмите **Сохранить**. Курс будет автоматически создан в статусе **Черновик**.
::

## Действия с курсами

В каждой строке таблицы доступны быстрые действия:

* **:icon{name="i-lucide-blocks"} Конструктор:** Открывает детальный редактор структуры, где вы управляете уроками и блоками контента.
* **:icon{name="i-lucide-pencil"} Редактировать:** Открывает панель свойств для изменения названия, описания или тегов.
* **:icon{name="i-lucide-trash"} Удалить:** Полностью удаляет курс.

::warning{icon="i-lucide-triangle-alert"}
**Удаление курса необратимо.** Вместе с курсом будут удалены все уроки, контентные блоки и данные о прогрессе студентов.
::

## Конструктор (Builder)

Клик по названию курса или иконке **Builder** перенаправляет вас в режим глубокого редактирования. Там вы сможете:
1.  Создавать **Уроки** (Lessons).
2.  Наполнять их **Блоками** (Текст, Видео, Код, Тесты).
3.  Настраивать **Флоу** (последовательное открытие материалов).

*(Подробная документация по Конструктору будет в следующем разделе).*
