# Getting Started

Athena LMS is a web-based learning management system designed to organize the educational process. This guide provides instructions on the initial system login and an overview of the main user interface.

Athena LMS is an open-source project. Source code is available on [GitHub](http://github.com/shekshuev/athena-lms).

## System Requirements

The system is a web application requiring a modern web browser with JavaScript enabled.

::warning{icon="i-lucide-triangle-alert"}
**Important:** Outdated browsers (e.g., Internet Explorer) are not supported.
::

  ::card{title="Google Chrome" icon="i-simple-icons-googlechrome" target="_blank" to="https://www.google.com/chrome/"}

## Authentication

Access is restricted to registered users. Accounts are created by administrators.

![Athena LMS Landing Page](/images/docs/en/landing-page-placeholder.png)

::steps{level="4"}
#### Navigate to Login
Go to the main page and click **START LEARNING** (or **Log in**).

#### Enter Credentials
You will be redirected to `/auth/login`. Fill in the fields:

::field-group
  :::field{name="Login" type="text" required="true"}
  Your unique identifier (not Email). Example: `smith_j`.
  :::
  :::field{name="Password" type="password" required="true"}
  Provided by the administrator.
  :::
::

#### Confirm
Click **Continue**.
::

::note{icon="i-lucide-info"}
If you forgot your password or see an *Invalid credentials* error, contact the administrator. Self-service reset is not available yet.
::

## Interface Overview

After logging in, the **Sidebar** becomes the main navigation tool.

::card-group
  ::card{title="Learn" icon="i-lucide-book-open"}
  For students. Courses, schedule, files, and community.
  ::
  ::card{title="Studio" icon="i-lucide-presentation"}
  For instructors. Course creation, grading, and analytics.
  ::
  ::card{title="Admin" icon="i-lucide-shield-check"}
  For administrators only. Manage users, roles, and system settings.
  ::
::

### User Menu

Located at the bottom of the sidebar (click on avatar):
* **Settings:** User profile.
* **Log out:** Securely end the session.

::caution{icon="i-lucide-shield"}
Always click **LOG OUT** when finishing your session on public computers.
::
