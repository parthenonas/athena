# Athena LMS Documentation

Welcome to the official documentation for **Athena LMS**, an open-source learning management system. This resource provides comprehensive information on configuring and using the platform.

::note{icon="i-simple-icons-github"}
Athena LMS is an Open Source project. Source code, changelogs, and issue tracking are available on [GitHub](https://github.com/shekshuev/athena-lms).
::
